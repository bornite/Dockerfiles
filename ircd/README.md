# A dockerfile for ircd(Internet Relay Chat Daemon)
====

## Description
Run simple ircd.

## Requirement

## Usage
- Build image
```
docker image build . -t simple-ircd
```

- Run daemon
```
docker run -d -p 6667:6667 simple-ircd:latest
```
